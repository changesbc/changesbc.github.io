03_使用海龟绘制图形

Turtle库是Python语言中一个很流行的绘制图像的函数库，想象一个小乌龟，在一个横轴为x、纵轴为y的坐标系原点，(0,0)位置开始，它根据一组函数指令的控制，在这个平面坐标系中移动，从而在它爬行的路径上绘制了图形。

一、导入库

  • import turtle
  • import turtle as t

二、基础指令

  • 隐藏海龟
  • 暂停程序，停止画笔绘制

作用是暂停程序，停止画笔绘制，但绘图窗体不关闭，直到用户关闭Python Turtle图形化窗口为止。

  • 显示海龟
  • 返回原点
  • 海龟速度
  • 如果输入数值大于 10 或小于 0.5 则速度设为0

六、设置朝向

角度 方向
0 右
90 上
180 左
270 下
  • 默认海龟朝向是朝右，即0度
  • 注意，角度为正，代表逆时针旋转
角度 方向
0 右
-90 下
-180 左
-270 上
  • 设置海龟方向为右下，角度为-45度
  • 角度为负数，代表是顺时针旋转
